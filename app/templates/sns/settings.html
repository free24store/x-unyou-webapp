SNS連携設定

X (Twitter)
  • テキスト投稿・画像・動画（MP4）に対応。
  • Developer Portal でアプリ作成 → Elevated または Basic access → Consumer Keys + Access Token を取得。
  • 注意: X利用規約の自動化ポリシーに注意。人間による投稿と区別できない自動投稿はリスクあり。
Instagram
  • Business/Creator アカウント + Meta Graph API 審査が必要。
  • 画像・動画はローカルファイルではなく公開URLが必要（S3・CDN等）。
TikTok
  • 縦型MP4動画の直接アップロードが可能（Content Posting API）。
  • デフォルトは「自分のみ（非公開）」で投稿されます。公開にするにはAPIパラメータを変更する必要あり。
YouTube
  • Google Cloud Console でプロジェクト作成 → YouTube Data API v3 有効化 → OAuth2設定。
  • デフォルトは「非公開」でアップロードされます。
  • アクセストークンの有効期限切れ時はリフレッシュトークンで自動更新されます。
設定の流れ: 各プラットフォームの「設定する」→ APIキー等を入力して保存 → 予約投稿画面で投稿先を選択
