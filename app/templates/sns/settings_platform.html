キャンセル
{% if platform == 'x' %}
X (Twitter) API設定手順:
1. Twitter Developer Portal でアプリを作成
2. Elevated access または Basic planが必要
3. 「User authentication settings」で Read and Write を許可
4. Consumer Keys と Access Token を生成してここに入力
{% elif platform == 'instagram' %}
Instagram API設定手順:
1. Meta for Developers でアプリを作成（Business タイプ）
2. Instagram Graph API のアクセス権限を申請
3. Page Access Token（長期トークン）を生成
4. Instagram Business Account ID を確認して入力
※ 画像/動画は公開URLが必要です（ローカルファイルは直接送れません）
{% elif platform == 'tiktok' %}
TikTok API設定手順:
1. TikTok for Developers でアプリを登録
2. Content Posting API の承認を取得
3. OAuth2 フローでアクセストークンを取得して入力
※ 投稿はデフォルト「自分のみ（非公開）」に設定されます
{% elif platform == 'youtube' %}
YouTube API設定手順:
1. Google Cloud Console でプロジェクトを作成
2. YouTube Data API v3 を有効化
3. OAuth2クライアントID（デスクトップアプリ）を作成
4. OAuth2認証フローを実行してアクセストークン・リフレッシュトークンを取得
※ アップロードはデフォルト「非公開」に設定されます
{% endif %} {% endblock %}
